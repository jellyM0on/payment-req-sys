# README

## Set up
Prerequisites: 
- Ruby
- Rails 
- Node 
- NPM / Yarn

1. Install backend dependencies: 
    - "bundle install" 
2. Set up database 
    - "rails db:create"
    - "rails db:migrate" 
    - "rails db:seed" to seed admin user 
3. Run api
    - "rails s" 
4. Install frontend dependencies: 
    - "cd ./front"
    - "npm install" 
5. Run frontend 
    - "npm run dev"
